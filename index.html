02
Introduction — CSS text-transform
What it is, why it matters, and how it works
🔒 Locked
🔒 Waiting for your teacher to unlock this activity. Sit tight!
The text-transform CSS property lets you control the capitalisation of text — making it ALL CAPS, all lowercase, or Capitalized — without changing the actual HTML content. Web developers use it to style headings, navigation links, and buttons consistently.
📐 Syntax
selector {
    text-transform: value;
}

/* Example */
p {
    text-transform: uppercase;
}
🎯 The Four Values
none
Input → "Hello World"
Hello World
No change. The browser shows the text exactly as written in HTML.
capitalize
Input → "hello world"
hello world
The first letter of every word is made capital.
uppercase
Input → "Hello World"
Hello World
ALL letters become CAPITAL letters.
lowercase
Input → "HELLO WORLD"
HELLO WORLD
all letters become small letters.
